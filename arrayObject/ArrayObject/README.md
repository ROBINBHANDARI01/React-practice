# Favorite Car Details App

A simple React app to practice **working with array objects** and **state**. Add your favorite cars and see them listed instantly!

---

## Features

- Add a car's **Year**, **Model**, and **Company**.
- Display all added cars in a list.
- Press **Enter** in the Year input to quickly add a car.
- Inputs clear automatically after submission.
